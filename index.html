secret santa

ruby on rails

The purpose for this little app is to simplify inter-organizational secret santas. Users register, add items to their wishlist, and can view the wishlists of their assigned giftee. An admin-level user manages relationships between users and assigns or reassigns secret santas as needed. The project is a fun look at self-joins, as it contains 4 separate many-to-many self-joins on the same user table.

This project was built using a vagrant box and running an ubuntu development environment. I used ruby on rails with a postresSQL database, and used bootstrap to style the front end. I deployed this project using AWS's ec2 platform.
